# neat_close
Initiates the closing procedure for a flow.


### Syntax

```c
neat_error_code neat_close(struct neat_ctx *ctx,
                           struct neat_flow *flow);
```

### Parameters

- **ctx**: Pointer to a NEAT context.
- **flow**: Pointer to the NEAT flow to be closed.

### Return values

- Returns `NEAT_OK`.
